Sistema de Gerenciamento de Usuários – API REST Spring Boot

API REST CRUD desenvolvida em Java com Spring Boot

Backend Java Spring Boot API REST

Propósito do Projeto

O Sistema de Gerenciamento de Usuários é uma API REST desenvolvida em Java utilizando Spring Boot com o objetivo de gerenciar registros de usuários em uma aplicação backend.

A API permite realizar operações completas de CRUD (Create, Read, Update e Delete), possibilitando o cadastro, consulta, atualização e remoção de usuários através de endpoints REST.

O projeto foi estruturado seguindo uma arquitetura em camadas, separando as responsabilidades entre Controller, Service e Repository, facilitando manutenção, organização do código e escalabilidade da aplicação.

Funcionalidades Chave

  • Cadastro de novos usuários na base de dados
  • Consulta de usuários através do email
  • Atualização de dados de usuários existentes
  • Remoção de usuários cadastrados
  • Integração com banco de dados relacional
  • API estruturada em arquitetura em camadas

Tecnologias Utilizadas

  • Java (JDK 17)
  • Spring Boot
  • Spring Web
  • Spring Data JPA
  • Arquitetura em camadas (Controller, Service, Repository)

Controller


    import com.diego.gerenciamento_usuario.business.UsuarioService;
    import com.diego.gerenciamento_usuario.infrastructure.entitys.Usuario;
    import jakarta.persistence.criteria.CriteriaBuilder;
    import lombok.RequiredArgsConstructor;
    import org.springframework.http.RequestEntity;
    import org.springframework.http.ResponseEntity;
    import org.springframework.web.bind.annotation.*;

    @RestController
    @RequestMapping("/usuario")
    @RequiredArgsConstructor
    public class UsuarioController {

        private final UsuarioService usuarioService;

        @PostMapping
        public ResponseEntity salvarUsuario(@RequestBody Usuario usuario){
            usuarioService.salvarUsuario(usuario);
            return ResponseEntity.ok().build();
        }

        @GetMapping
        public ResponseEntity buscarUsuarioPorEmail(@RequestParam String email){
            return ResponseEntity.ok(usuarioService.buscarUsuarioPorEmail(email));
        }

        @DeleteMapping
        public ResponseEntity deletarUsuarioPorEmail(@RequestParam String email){
            usuarioService.deletarUsuarioPorEmail(email);
            return ResponseEntity.ok().build();
        }

        @PutMapping
        public ResponseEntity atualizarUsuarioPorId(@RequestParam Integer id, @RequestBody Usuario usuario){

            usuarioService.atualizarUsuarioPorId(id, usuario);
            return ResponseEntity.ok().build();
        }

    }
