ShopEase

Gestão colaborativa de compras em tempo real.

Android Java Firebase NoSQL

Propósito do Projeto

O ShopEase nasceu da necessidade de organizar compras domésticas entre múltiplos usuários. Muitas vezes, as pessoas compram itens repetidos ou esquecem de algo porque a comunicação via mensagens é fragmentada. Este app centraliza a lista em tempo real.

O que aprendi

Durante o desenvolvimento, aprofundei meus conhecimentos em sincronização de dados assíncronos com Firebase Realtime Database. Aprendi a lidar com o ciclo de vida do Android para garantir que a interface não travasse durante atualizações pesadas e implementei um sistema de autenticação seguro.

Funcionalidades Chave

  • Criação de listas compartilhadas com sistema de grupos fechados
  • Atualização instantânea (Realtime) em todo o App
  • Produtos separados em categorias e com sistema de busca
  • Adição de produtos totalmente personalizados

Tecnologias Utilizadas

  • Java (JDK 17)
  • Firebase Realtime DB
  • Firebase Auth
  • Arquitetura MVVM
